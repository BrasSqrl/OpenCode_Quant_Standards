# Experiment Note Template

Use this template when the user asks for an experiment plan, benchmark summary, or backtest writeup.

## Hypothesis

- What is being tested and why it should matter

## Setup

- Dataset window
- Universe or segment
- Target and horizon
- Feature set or change under test
- Baselines
- Evaluation protocol

## Results

- Headline metrics
- Comparison to baseline
- Stability across key slices or periods
- Important plots or artifacts

## Interpretation

- What the evidence supports
- What it does not support
- Main caveats or threats to validity

## Decision

- Continue
- Revise
- Stop

Include the exact reason for the decision and the next step.
