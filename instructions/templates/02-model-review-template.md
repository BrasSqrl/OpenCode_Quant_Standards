# Model Review Template

Use this template when the user asks for a formal model review, validation memo, or approval-oriented assessment.

## Review Scope

- Model or artifact under review
- Date or code version reviewed
- Artifacts examined
- Review lens: methodology, implementation, controls, production readiness

## Decision Summary

Choose one:

- Approve
- Approve with conditions
- Hold pending evidence
- Reject

Include a short reason tied to evidence.

## Material Findings

For each finding, include:

- Title
- Severity
- Evidence
- Why it matters
- Required action

## Methodology Assessment

- Problem definition
- Target and horizon clarity
- Data availability and leakage risk
- Validation design
- Baselines and benchmarks
- Stability and robustness

## Implementation Assessment

- Reproducibility
- Test coverage or verification quality
- Configuration clarity
- Data and time handling
- Operational risks

## Residual Risks And Required Next Steps

- Open risks that remain even if the current issues are fixed
- Minimum evidence needed to move to the next approval state
